# Copper Chimney

A premium restaurant website template.

## Features

- Fully Responsive (Mobile First Design)
- Interactive 3D Hero Section
- Smooth Animations (GSAP)
- Dedicated About Us and Menu pages
- Booking Modal

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Three.js
- GSAP

## Run Clean

To run the project locally, simply open `index.html` in your browser.
